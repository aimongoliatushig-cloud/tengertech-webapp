1. Dashboard

Ажлын самбар ашиглах

Нэвтэрсний дараа хамгийн түрүүнд харах дэлгэц.

1Нийт ажил, хянах ажил, хугацаа хэтэрсэн тоог шалгана.
2Хурдан холбоосоор шинэ ажил, жагсаалт, тайлан, календарь руу орно.
3Role-оос хамаарч бүх байгууллага эсвэл зөвхөн өөрийн хэлтэс харагдана.
1
2. Нэмэх төв

Юу нэмэхээ сонгох

1Sidebar дээр “Шинэ ажил” дарна.
2Ажил нэмэх, даалгавар нэмэх, тайлан оруулах урсгалаас сонгоно.
3Таны role-д зөвшөөрөгдсөн сонголтууд л харагдана.
2
3. Шинэ ажил

Ажил үүсгэх

1Хэлтэс сонгоно. Хэлтсийн дарга/мастер дээр автоматаар түгжигдэж болно.
2Ерөнхий ажил бол нэр, хариуцагч, төрөл, хугацаа оруулна.
3Хог тээвэрлэлт бол машин, маршрут, огноо сонгоно.
4“Ажил үүсгэх” дарж Odoo руу бүртгэнэ.
3
4. Календарь

Ажлыг өдөр/долоо хоног/сараар харах

1Календарь руу орно.
2Долоо хоног эсвэл сарын view сонгоно.
3Card дээр дарахад тухайн даалгаврын дэлгэрэнгүй рүү орно.
4
5. Мэдэгдэл

Өөрт ирсэн ажил, хяналт харах

1Header дээрх bell icon эсвэл sidebar “Мэдэгдэл” дээр дарна.
2Шинэ ажил, хянах ажил, хугацаа хэтэрсэн анхааруулга харагдана.
3Мэдэгдэл дээр дарж тухайн ажил/даалгавар руу орно.

Browser notification popup гарвал Allow дарна. Өмнө нь Block хийсэн бол Chrome Site settings дээр Allow болгож refresh хийнэ.

5
6. Тайлан

Тайлан статистик ашиглах

1Хэлтэс болон доторх нэгжээ сонгоно.
2Хог тээвэрлэлт дээр машины өдөр тутмын жин, нийт кг харагдана.
3Өнөөдрийн явц: явж буй, хяналт хүлээж буй, дууссан ажлыг тусад нь харна.
6
7. Тохиргоо

Баг, маршрут, хогийн цэг

1Профайл/Тохиргоо хэсэг рүү орно.
2Баг үүсгэхдээ багийн нэр болон багийн гишүүдийг сонгоно.
3Хог ачилт/Авто баазын эрхтэй хэрэглэгч маршрут, хогийн цэгийн тохиргоо үүсгэнэ.
