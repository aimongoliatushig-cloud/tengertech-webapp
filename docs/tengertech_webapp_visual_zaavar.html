Тусламжийн шинэ бүтэц

Нэвтэрсэн хүний role-д тохирсон зурагтай заавар

App доторх /help хэсэг хэрэглэгчийн role, group flags, хэлтсийн хүрээгээр зөвхөн тухайн хүнд хэрэгтэй сэдвүүдийг харуулна. Хайлт дээр “тайлан”, “зураг”, “батлах”, “машин”, “HR” гэх мэт үг бичихэд тохирох заавар clickable байдлаар гарна.

Энэ HTML нь сургалт, танилцуулгад зориулсан offline хувилбар. Бодит app дээрх заавар login хийсэн хэрэглэгчийн эрхээр автоматаар шүүгдэнэ.

Role matrix

Role тус бүрийн харах тусламж

CEO / Захирал / Ерөнхий менежер

Ерөнхий хяналт, бүх алба, тайлан, худалдан авалт, авто бааз, HR зэрэг олон сэдэв харагдана.

Хэлтсийн дарга

Өөрийн хэлтсийн ажил үүсгэх, хянах, батлах, буцаах, тайлан харах сэдвүүд гарна.

Ахлах мастер / Мастер

Өөрт болон багт оноогдсон ажил, тайлан мэдээ оруулах, зураг хавсаргах, буцаагдсан тайлан засах заавар гарна.

Ажилтан / Field user

Зөвхөн өөрт оноогдсон ажил, mobile тайлан илгээх, мэдэгдэл, профайл зэрэг minimum урсгал харагдана.

HR

Ажилтан, чөлөө, өвчтэй, сахилга, шилжилт хөдөлгөөний бүртгэлийн тусламж гарна.

Авто бааз / Худалдан авалт

Засварын хүсэлт, машин, үнийн санал, санхүү, төлбөр, шийдвэрийн урсгал харагдана.

Screen заавар

Гол дэлгэцүүд дээр хаана дарах вэ

  1. 1“Хяналтын самбар” дарна. Role-д зөвшөөрөгдсөн ажил, хэлтэс, тайлангийн товч мэдээлэл гарна.
  2. 2Тоон картуудыг шалгана. Хугацаа хэтэрсэн, хяналт хүлээж буй, явж буй ажлын тоог харна.
  3. 3Card дээр дарна. Тухайн ажил, жагсаалт, тайлан эсвэл мэдэгдлийн дэлгэц рүү орно.
  1. 1Зүүн цэсний “Шинэ ажил” дарна. Энэ товч зөвхөн эрхтэй хэрэглэгчид харагдана.
  2. 2Үүсгэх урсгалаа сонгоно. Ажил, даалгавар, тайлангийн зөв төрлийг дарна.
  3. 3Мэдээллээ бөглөөд хадгална. Хэлтэс, хариуцагч, хугацаа, тайлбараа оруулна.
  1. 1Доод цэснээс “Ажил” дарна. Зөвхөн өөрт оноогдсон ажил харагдана.
  2. 2Ажлын card дээр дарна. Байршил, хугацаа, тайлангийн хэсэг нээгдэнэ.
  3. 3Зураг хавсаргаад “Тайлан илгээх” дарна. Тайлан хяналт руу шилжинэ.
  1. 1“Мэдэгдэл” дарна. Шинэ ажил, буцаагдсан тайлан, хугацаа хэтэрсэн анхааруулга гарна.
  2. 2Мэдэгдлийн card дээр дарна. Холбогдох ажил эсвэл тайлан руу шууд орно.
  3. 3Өөрийн үйлдлээ дуусгана. Ажилтан засаж илгээнэ, дарга батлах эсвэл буцаана.
