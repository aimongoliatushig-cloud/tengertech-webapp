# QA Asset Index

This folder stores UI acceptance screenshots and mockup references for the municipal ERP web app.

Current assets:

- `desktop-dashboard-preview.png` - desktop dashboard reference copied from the existing public preview asset.
- `mobile-dashboard-preview.png` - mobile dashboard reference copied from the existing public preview asset.
- `municipal-design-board-showcase.png` - design board mockup reference copied from the existing public showcase asset.

When adding new evidence:

- Use descriptive lowercase filenames.
- Keep screenshots tied to the flow name, for example `mobile-worker-field-flow.png`.
- Do not store credentials, production data, or private citizen/employee information in screenshots.
