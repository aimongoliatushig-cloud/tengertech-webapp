# QA Asset Index

This folder stores UI acceptance screenshots and mockup references for the municipal ERP web app.

Current assets:

- `desktop-dashboard-preview.png` - desktop dashboard reference copied from the existing public preview asset.
- `mobile-dashboard-preview.png` - mobile dashboard reference copied from the existing public preview asset.
- `municipal-design-board-showcase.png` - design board mockup reference copied from the existing public showcase asset.
- `mobile-worker-assigned.png` - mobile worker assigned-work screenshot.
- `manager-tohijilt-desktop.png` - department manager desktop screenshot.
- `inspector-master-desktop.png` - inspector/master desktop screenshot.
- `hr-dashboard-desktop.png` - HR dashboard desktop screenshot.
- `hr-dashboard-mobile.png` - HR dashboard mobile screenshot.
- `garbage-route-today-desktop.png` - garbage route today-view desktop screenshot.
- `garbage-route-weekly-plan-mobile.png` - garbage route weekly-plan mobile screenshot.
- `repair-requests-list.png` - repair request list screenshot.
- `repair-request-detail.png` - repair request detail screenshot.
- `role-ui-summary.json` - role UI screenshot run summary.
- `permission-matrix.json` - role permission reference from QA output.

When adding new evidence:

- Use descriptive lowercase filenames.
- Keep screenshots tied to the flow name, for example `mobile-worker-field-flow.png`.
- Do not store credentials, production data, or private citizen/employee information in screenshots.
